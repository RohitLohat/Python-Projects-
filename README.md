# HR Diagram Plotter

This project plots a simple Hertzsprung-Russell (HR) diagram using Python.  
It visualizes stars by temperature and luminosity, showing different spectral types.  

## How to Run
1. Install Python libraries: matplotlib, numpy  
2. Run the file `hr_diagram.py` in Python environment or Pydroid  
3. View the plotted HR diagram  

## Features
- Plots Temperature vs Luminosity  
- Inverts X-axis for hot stars on the left  
- Uses logarithmic scale for luminosity  
- Labels different star types (O, B, A, G, K, M)
